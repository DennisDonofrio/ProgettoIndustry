# Sistema di Monitoraggio Empirico con LEGO Mindstorms

## Descrizione del Progetto

Questo progetto nasce con l’obiettivo di realizzare un sistema di monitoraggio utilizzando la piattaforma LEGO Mindstorms.  
Il sistema raccoglie dati da diversi sensori e permette di osservare il comportamento di un ambiente o di un processo in modo empirico e sperimentale.

L’idea principale è utilizzare componenti semplici e programmabili per creare un sistema modulare, facilmente modificabile e adatto ad attività didattiche, sperimentazioni e prototipazione rapida.

---

## Obiettivi

- Acquisire dati tramite sensori LEGO Mindstorms
- Monitorare parametri in tempo reale
- Analizzare i dati raccolti
- Sperimentare approcci empirici al monitoraggio
- Sviluppare competenze di programmazione e integrazione hardware/software

---

## Tecnologie Utilizzate

- LEGO Mindstorms
- Sensori LEGO (luce, distanza, colore, temperatura, ecc.)
- Python / software LEGO Mindstorms
- Git & GitHub per il versionamento del progetto

---

## Funzionalità

- Lettura dei dati dai sensori
- Raccolta e visualizzazione delle misurazioni
- Logging dei dati
- Struttura modulare per aggiungere nuovi sensori
- Approccio sperimentale basato su test empirici
